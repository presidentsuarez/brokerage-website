# brokerage-website
Public marketing & landing-page site for Realty ONE Group Advantage (Prism).
Static, brand-matched, deployed via GitHub Pages. Independent of the app repo.

- `index.html` — main landing page
- `join.html` — agent recruiting landing (template to clone for new campaigns)
- `assets/` — shared brand styles + logo
Add a new landing page by copying `join.html` to `<campaign>.html`.
